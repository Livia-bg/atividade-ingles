 ## Glossary
  Install:(instalar) colocar algo em um lugar para funcionar ou estabelecer alguém em um novo local.
  Npm:(Node Package Manger) é uma ferramenta essencial para desenvolvedores que trabalham com o Node.js. Ela permite que os usuários instalem, atualizem e gerenciem bibliotecas e dependências de forma eficiente. 
  Dev:(desenvolvedor):é um profissional essencial no mundo digital, responsável por projetar, codificar, testar e manter softwares.
  Cd(Cade):achar pasta
  Code(código): O código associa cada letra a uma palavra que se inicia com a letra em questão.
  Git é um sistema de controle de versão distribuído amplamente utilizado no desenvolvimento de software. 
  Run(correr): esse verbo pode ter outros significados dependendo do contexto, como funcionar ou executar. 
  LocalHost(localhost):local de hospedagem
  Readme(leia-me):Ele fornece informações essenciais sobre o projeto, instruções de para instalar.
  Package(Pacote): organização de pastas dentro do projeto para facilitar o gerenciamento e a navegação pelo código.
